# Kisan Center

Kisan Center is a specialized e-commerce platform designed to provide farmers and agricultural enthusiasts with easy access to high-quality farming products and resources. Built with modern web technologies, it offers a seamless shopping experience tailored for the agricultural community.

## 🌟 Key Features

- **Product Showcase**: High-resolution product displays with zoom functionality (`react-inner-image-zoom`).
- **Interactive Carousels**: Smooth product and banner sliders using `react-slick`.
- **Responsive Design**: Fully mobile-responsive interface built with Bootstrap 5.
- **State Management**: Robust application state handling via Redux Toolkit.
- **Real-time Notifications**: Dynamic user feedback using React Toastify.
- **API Integration**: Efficient data fetching and management with Axios.
- **Iconography**: Extensive use of Font Awesome for a clean, intuitive UI.

## 🛠️ Technology Stack

- **Frontend Framework**: [React](https://reactjs.org/) (v19)
- **Styling**: [Bootstrap](https://getbootstrap.com/) (v5.3.3) & Custom CSS
- **State Management**: [Redux Toolkit](https://redux-toolkit.js.org/) & React-Redux
- **Routing**: [React Router DOM](https://reactrouter.com/)
- **Icons**: [Font Awesome](https://fontawesome.com/)
- **HTTP Client**: [Axios](https://axios-http.com/)
- **Components**: Slick Carousel, React Inner Image Zoom, React Toastify

## 🚀 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) (v16+ recommended) installed.

### Installation

1. Clone or download the project files.
2. Navigate to the project directory:
   ```bash
   cd kisan-center-static
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the application in development mode:

```bash
npm start
```
The app will run at [http://localhost:3000](http://localhost:3000).

### Building for Production

To create an optimized production build:

```bash
npm run build
```
The output will be in the `build` folder.

## 📂 Project Structure

- `src/`
  - `components/` - Reusable UI components.
  - `pages/` - Main page views.
  - `store/` - Redux store configuration and slices.
  - `assets/` - Images, styles, and other static files.
- `public/` - Static assets and index.html.
